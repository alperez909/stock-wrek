# Stock Wrek

Stock Wrek makes it easy to follow your favorite stocks.
## Getting Started

Install and run the app in the development mode.

```sh
cd /path/to/stock-wrek
npm install
npm start
```

Open http://localhost:3000 to view it in your browser.
